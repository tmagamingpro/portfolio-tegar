# Portfolio Website

Proyek sederhana berisi halaman statis untuk portofolio pribadi.

Struktur folder:

- `index.html` - Halaman utama (landing page).
- `about.html` - Halaman tentang/biografi.
- `contact.html` - Halaman kontak.
- `project.html` - Halaman yang menampilkan proyek-proyek.
- `skills.html` - Halaman yang menampilkan keterampilan.
- `css/` - Folder berisi stylesheet:
  - `style.css` - Gaya utama situs.
- `img/` - Folder untuk menyimpan gambar (foto, screenshot proyek, ikon, dll.).
- `js/` - Folder untuk file JavaScript:
  - `script.js` - Script frontend (interaksi sederhana).

Cara menjalankan:

1. Buka file `index.html` di browser (klik dua kali atau "Open with" > browser).
2. Jika ingin mengembangkan, edit file HTML/CSS/JS secara langsung dan refresh browser.

Panduan singkat pengembangan:

- Gunakan editor teks (VS Code) untuk mengedit file.
- Perubahan CSS ada di `css/style.css`.
- Tambah/ubah gambar di `img/` dan referensikan dari HTML.
- Tambah interaksi atau fungsionalitas di `js/script.js`.

Kontribusi:

- Untuk pembaruan kecil, buat cabang (branch) baru, lakukan perubahan, lalu buat pull request.

Lisensi:

- Tidak ada lisensi resmi di repositori ini. Tambahkan file `LICENSE` jika diperlukan.

Catatan:

- Situs ini bersifat statis dan tidak memerlukan server backend; hosting bisa menggunakan GitHub Pages atau layanan hosting statis lainnya.
